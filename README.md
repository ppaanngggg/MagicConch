## How to install

1. install [wails](https://github.com/wailsapp/wails)

```bash
go install github.com/wailsapp/wails/v2/cmd/wails@latest
```

2. check dependencies

```bash
wails doctor
```

3. build binary

```bash
wails build
```

4. find binary file in `build/bin`
